Cards - my master work

Cards is a response to the lack of viable software options for implementing an efficient, out-of-the-way GTD system.

I couldn’t find a great solution so I’m building it myself.

After building a crude version in OSX, I’m porting the project to a web-app built on AngularJS and Bootstrap.
