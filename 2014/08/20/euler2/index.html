Project Euler 2 - Even Fibonacci numbers

Aug 20, 2014

Problem 2

We need to compute the sum of even numbers in the Fibonacci sequence below 4 million.

It’s a pretty straightforward algorithm that utilizes a while loop to compute the next number in the sequence and check if it’s even. 

 1 def computeEvenFibonacciNumbersBelow(bigNumber):
 2     
 3     # declare first fibonacci numbers
 4     fibA = 0
 5     fibB = 1
 6     
 7     # declare our running sum
 8     sumOfEvens = 0
 9     
10     # function to calculate next number in fib sequence
11     def nextFib():
12         return fibA + fibB
13     
14     # run loop until the fib sequence has surpassed our limit
15     while fibB < bigNumber:
16         
17         # compute next fibonacci number
18         x = nextFib()
19         
20         # double check we haven't exceeded the bigNumber
21         # add to sum if number is even
22         if x < bigNumber:
23             if x % 2 == 0:
24                 sumOfEvens += x
25         
26         # update our two fib numbers
27         fibA = fibB
28         fibB = x
29         
30     return sumOfEvens

Consequently, computeEvenFibonacciNumbersBelow(4000000) = 4613732.

Interestingly, the sum of odds below 4000000 equals the sum of evens. This makes intuitive sense, given that every even number in the sequence is the sum of the two digits before it, which are odd. Each odd number (aside from the first 1) is the sum of the preceding two numbers, one of which is odd, the other even.

And thus, we have a rule: the sum of odd Fibonacci numbers equals the sum of even Fibonacci numbers below limit N, as long as the largest number in the Fibonacci sequence below that limit is even.
